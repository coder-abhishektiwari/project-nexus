# Project Nexus

Project Nexus is a web-based project marketplace platform designed for students to discover, download, and sell academic projects. The platform connects students who need projects for their college work with those who want to share or sell their own projects.

The system includes role-based authentication with separate flows for **Admin and Users**. Users can browse available projects, download them for academic use, or become sellers and upload their own projects.

The platform uses **Supabase** for backend services including authentication and database management, and **Razorpay** for secure online payments when purchasing projects.

## Tech Stack

* React
* TypeScript
* Tailwind CSS
* Supabase
* Razorpay
* Vite

# Project Nexus

Project Nexus is a web-based project marketplace platform designed for students to find, download, and sell academic projects. The platform connects students who need projects for their college work with those who want to share or sell their own projects.

It includes role-based authentication with **Admin and User flows**. Users can browse projects, download them for academic use, or become sellers and upload their own projects.

The platform uses **Supabase** for backend services and authentication, and **Razorpay** for secure project purchase payments.
